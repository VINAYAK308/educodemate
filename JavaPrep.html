Introduction to Java

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is a general-purpose programming language intended to let application developers write once, run anywhere (WORA), meaning that compiled Java code can run on all platforms that support Java without the need for recompilation.

Basics Structure of Java Program

A Java program typically consists of the following parts:

  1. Package Declaration: Defines the package to which the class belongs.
  2. Import Statements: Imports other Java packages or classes.
  3. Class Declaration: Defines a class.
  4. Main Method: Entry point of the program.
  5. Statements: Actual code statements.

Example: Hello World Program

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

This program prints "Hello, World!" to the console.

Explanation

  • public class HelloWorld: Declares a public class named HelloWorld.
  • public static void main(String[] args): The main method where the program execution begins.
  • System.out.println("Hello, World!");: Outputs "Hello, World!" followed by a newline to the console.

Variables and Data Types

In Java, a variable is a named storage location that holds a value of a specific data type. Variables must be declared before they can be used. Here are some key points:

  • Declaration: Variables are declared with a data type followed by a name.
  • Initialization: Variables can be initialized (assigned an initial value) at the time of declaration.
  • Scope: Variables have a scope that determines where they can be accessed.
  • Types: Java supports various data types such as int, float, double, char, boolean, etc.

Example:

public class VariableExample {
    public static void main(String[] args) {
        // Variable declaration and initialization
        int age = 25;
        double height = 1.75;
        char grade = 'A';
        boolean isStudent = true;

        // Output variables
        System.out.println("Age: " + age);
        System.out.println("Height: " + height + " meters");
        System.out.println("Grade: " + grade);
        System.out.println("Is student: " + (isStudent ? "Yes" : "No"));
    }
}

This program declares and initializes variables of different data types and outputs their values.

Control Structures

If Statement

The if statement executes a block of code if a specified condition is true.

public class IfExample {
    public static void main(String[] args) {
        int num = 10;

        // If statement
        if (num > 0) {
            System.out.println("Number is positive");
        }
    }
}

This program checks if a number is positive and prints a message accordingly.

If-Else Statement

The if-else statement executes one block of code if a condition is true and another block if the condition is false.

public class IfElseExample {
    public static void main(String[] args) {
        int num = -5;

        // If-Else statement
        if (num > 0) {
            System.out.println("Number is positive");
        } else {
            System.out.println("Number is not positive");
        }
    }
}

This program checks if a number is positive or not and prints the respective message.

Switch Statement

The switch statement allows a variable to be tested for equality against a list of values.

public class SwitchExample {
    public static void main(String[] args) {
        int choice = 2;

        // Switch statement
        switch (choice) {
            case 1:
                System.out.println("Choice is 1");
                break;
            case 2:
                System.out.println("Choice is 2");
                break;
            case 3:
                System.out.println("Choice is 3");
                break;
            default:
                System.out.println("Invalid choice");
        }
    }
}

This program demonstrates a switch statement based on the value of the variable choice.

Loops

Java supports three types of loops: for, while, and do-while.

For Loop

public class ForLoopExample {
    public static void main(String[] args) {
        // For loop example
        for (int i = 1; i <= 5; ++i) {
            System.out.print(i + " ");
        }
        System.out.println();
    }
}

This program uses a for loop to print numbers from 1 to 5.

While Loop

public class WhileLoopExample {
    public static void main(String[] args) {
        int num = 1;

        // While loop example
        while (num <= 5) {
            System.out.print(num + " ");
            ++num;
        }
        System.out.println();
    }
}

This program uses a while loop to print numbers from 1 to 5.

Do-While Loop

public class DoWhileLoopExample {
    public static void main(String[] args) {
        int num = 1;

        // Do-While loop example
        do {
            System.out.print(num + " ");
            ++num;
        } while (num <= 5);
        System.out.println();
    }
}

This program uses a do-while loop to print numbers from 1 to 5.

Methods

Method Declaration

A method in Java is a block of code that performs a specific task. It consists of a method header and a method body:

return_type method_name(parameters) {
    // Method body
}

Example:

public class MethodExample {
    // Method declaration
    static void greet() {
        System.out.println("Hello, welcome to Java!");
    }

    public static void main(String[] args) {
        // Method call
        greet();
    }
}

This program defines and calls a method named greet that prints a greeting message.

Parameters and Return Types

Methods can accept parameters and return values:

public class MethodExample {
    // Method with parameters and return type
    static int add(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        // Method call
        int result = add(5, 3);
        System.out.println("Sum: " + result);
    }
}

This program defines a method add that accepts two integers, adds them, and returns the sum.

Classes and Objects

Class Declaration

A class is a blueprint for creating objects. It contains fields (variables) and methods to define the behavior of objects:

class ClassName {
    // Fields
    // Methods
}

Example:

public class Person {
    // Fields
    String name;
    int age;

    // Method
    void display() {
        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
    }

    public static void main(String[] args) {
        // Create an object
        Person p = new Person();

        // Initialize fields
        p.name = "John";
        p.age = 30;

        // Call method
        p.display();
    }
}

This program defines a Person class with fields name and age, and a method display to print the values.

Constructors

A constructor is a special method that is called when an object is instantiated. It is used to initialize the object's state:

class ClassName {
    // Constructor
    ClassName() {
        // Initialization code
    }
}

Example:

public class Person {
    String name;
    int age;

    // Constructor
    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    void display() {
        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
    }

    public static void main(String[] args) {
        // Create an object using constructor
        Person p = new Person("John", 30);

        // Call method
        p.display();
    }
}

This program defines a constructor for the Person class to initialize the fields.

Inheritance

Inheritance allows a class to inherit properties and behavior from another class:

class BaseClass {
    // Fields and methods
}

class DerivedClass extends BaseClass {
    // Additional fields and methods
}

Example:

class Animal {
    void eat() {
        System.out.println("This animal eats food.");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("The dog barks.");
    }
}

public class InheritanceExample {
    public static void main(String[] args) {
        Dog d = new Dog();
        d.eat(); // Inherited method
        d.bark(); // Dog's own method
    }
}

This program demonstrates inheritance where the Dog class inherits the eat method from the Animal class.

Exceptions

Exceptions are events that disrupt the normal flow of a program. Java provides a robust mechanism to handle exceptions:

Try-Catch Block

The try-catch block is used to handle exceptions:

try {
    // Code that may throw an exception
} catch (ExceptionType e) {
    // Handle exception
}

Example:

public class ExceptionExample {
    public static void main(String[] args) {
        try {
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            System.out.println("ArithmeticException: " + e.getMessage());
        }
    }
}

This program demonstrates exception handling using a try-catch block to catch an ArithmeticException.

Finally Block

The finally block contains code that is always executed, regardless of whether an exception is thrown:

try {
    // Code that may throw an exception
} catch (ExceptionType e) {
    // Handle exception
} finally {
    // Code that will always be executed
}

Example:

public class FinallyExample {
    public static void main(String[] args) {
        try {
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            System.out.println("ArithmeticException: " + e.getMessage());
        } finally {
            System.out.println("Finally block is always executed.");
        }
    }
}

This program demonstrates the use of a finally block, which is executed after the try-catch block.

Throw and Throws

The throw keyword is used to explicitly throw an exception, while the throws keyword is used in method signatures to declare that a method may throw exceptions:

class CustomException extends Exception {
    CustomException(String message) {
        super(message);
    }
}

public class ThrowThrowsExample {
    // Method declaring it may throw an exception
    static void validate(int age) throws CustomException {
        if (age < 18) {
            throw new CustomException("Age is less than 18");
        } else {
            System.out.println("Age is valid");
        }
    }

    public static void main(String[] args) {
        try {
            validate(15);
        } catch (CustomException e) {
            System.out.println("CustomException: " + e.getMessage());
        }
    }
}

This program demonstrates the use of the throw and throws keywords with a custom exception.

File I/O

Java provides a robust I/O package to handle input and output operations with files:

Writing to a File

The FileWriter class is used to write data to a file:

import java.io.FileWriter;
import java.io.IOException;

public class FileWriteExample {
    public static void main(String[] args) {
        try (FileWriter writer = new FileWriter("output.txt")) {
            writer.write("Hello, FileWriter!");
        } catch (IOException e) {
            System.out.println("IOException: " + e.getMessage());
        }
    }
}

This program writes the text "Hello, FileWriter!" to a file named output.txt.

Reading from a File

The FileReader class is used to read data from a file:

import java.io.FileReader;
import java.io.IOException;

public class FileReadExample {
    public static void main(String[] args) {
        try (FileReader reader = new FileReader("output.txt")) {
            int character;
            while ((character = reader.read()) != -1) {
                System.out.print((char) character);
            }
        } catch (IOException e) {
            System.out.println("IOException: " + e.getMessage());
        }
    }
}

This program reads and prints the contents of the file named output.txt.

Advanced Topics

Multithreading

Java supports multithreading, which allows concurrent execution of two or more threads:

class MyThread extends Thread {
    public void run() {
        for (int i = 0; i < 5; i++) {
            System.out.println(i + " " + Thread.currentThread().getName());
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                System.out.println(e);
            }
        }
    }

    public static void main(String[] args) {
        MyThread t1 = new MyThread();
        MyThread t2 = new MyThread();
        t1.start();
        t2.start();
    }
}

This program demonstrates the creation and execution of two threads that print numbers from 0 to 4.

Collections

Java provides a comprehensive set of collection classes for storing and manipulating groups of data:

import java.util.ArrayList;
import java.util.List;

public class CollectionExample {
    public static void main(String[] args) {
        List<String> list = new ArrayList<>();
        list.add("Java");
        list.add("Python");
        list.add("C++");

        for (String lang : list) {
            System.out.println(lang);
        }
    }
}

This program demonstrates the use of an ArrayList to store and print a list of programming languages.

Lambda Expressions

Lambda expressions in Java provide a clear and concise way to represent one method interface using an expression:

import java.util.Arrays;
import java.util.List;

public class LambdaExample {
    public static void main(String[] args) {
        List<String> list = Arrays.asList("Java", "Python", "C++");

        // Lambda expression
        list.forEach(lang -> System.out.println(lang));
    }
}

This program uses a lambda expression to print elements of a list.
